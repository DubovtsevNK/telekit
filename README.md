# Telekit

A Go library for building Telegram bots with state machine support.

Features
 State Machine — Built-in state machine for managing multi-step dialogues
 Scenarios — Define complex conversation flows with ease
 Input Validation — Validate user input at each step
 Callback Support — Handle inline button callbacks with data parsing
 Command Handling — Register and handle bot commands
 Logging — Structured logging with Zap
 Lightweight — Minimal dependencies, easy to integrate


Telekit — это библиотека для Go, которая упрощает создание Telegram-ботов 
со сложными сценариями взаимодействия. Встроенная state-машина позволяет 
легко управлять многошаговыми диалогами с пользователями.

Особенности

 State-машина для управления сценариями
 Поддержка команд, callback'ов и сообщений
 Валидация пользовательского ввода
 Простая регистрация обработчиков
 Логирование через zap
